# Система контроля и планирования ремонта МВПС

Сервис для управления ремонтными заданиями моторвагонного подвижного состава, планирования ТО/ТР, а также контроля работы вспомогательных бригад и расхода запчастей.

## 1. Системные требования

Для успешного развертывания инфраструктуры на хост-машине (сервере) должны быть установлены:
* **Операционная система:** Linux, macOS или Windows (через WSL2).
* **Docker:** Engine версии 20.10.0 и выше.
* **Docker Compose:** плагин версии v2.0.0 и выше.
* **Сеть:** Свободные порты `8000` (приложение/панель управления) и `5432` (PostgreSQL).

## 2. Структура конфигурационных файлов

Проект использует переменные окружения для настройки базы данных и параметров безопасности. Перед запуском необходимо создать файл `.env` в корневой директории репозитория.

**Пример конфигурации (`.env`):**
```ini
# Настройки подключения к СУБД PostgreSQL
DB_USER=postgres
DB_PASSWORD=your_strong_password
DB_HOST=db
DB_PORT=5432
DB_NAME=vkr_repair_db

# Секретный ключ для подписи токенов и сессий авторизации (SQLAdmin)
SECRET_KEY=your_super_secret_key_here
